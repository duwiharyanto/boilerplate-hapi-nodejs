## Deployment

Clone terlebih dahulu
```bash
  git clone <URL_REPOSITORI>
```
Masuk ke folder proyek
```bash
  cd nama-proyek
```
Install dependency
```bash
  npm run install
```
Menjalankan aplikasi
```bash
  npm run start:dev
```
